# AUTOMATIZĒTAS TESTĒŠANAS PLATFORMU SALĪDZINOŠA ANALĪZE STUDENTU PROGRAMMU PĀRBAUDES KONTEKSTĀ. 

Šajā repozitorijā apkopoti bakalaura darba praktiskajā aprobācijā izmantotie materiāli automatizētas testēšanas platformu salīdzinošai analīzei studentu C++ programmu pārbaudes kontekstā.

Darba tēma: **“Automatizētas testēšanas platformu salīdzinoša analīze studentu programmu pārbaudes kontekstā”**.

Repozitorijā ir iekļauti praktiskās pārbaudes materiāli: anonimizētu studentu darbu arhīvs, pārbaudes skripti, ievades dati, konfigurācijas faili, platformu testēšanas piemēri. Šie materiāli izmantoti Web-CAT, GitLab CI, CodeGrade un GitHub Classroom platformu praktiskai salīdzināšanai RTU kursa “Objektorientētā programmēšana” (DPI230) kontekstā.
